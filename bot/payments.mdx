</Note>

## YooKassa

Самый популярный провайдер для российского рынка. Поддерживает банковские карты и СБП.

```env
YOOKASSA_SHOP_ID=123456
YOOKASSA_SECRET_KEY=live_xxxxxxxxxxxxx
YOOKASSA_RETURN_URL=https://t.me/your_bot
YOOKASSA_SBP_ENABLED=true
YOOKASSA_DEFAULT_RECEIPT_EMAIL=receipts@example.com
YOOKASSA_VAT_CODE=1
```

Webhook: `https://hooks.domain.com/yookassa-webhook`

<Info>
  Бот проверяет IP-адреса YooKassa и отклоняет запросы с неизвестных адресов.
</Info>

### Лимиты

| Параметр | Переменная | По умолчанию |
| -------- | ---------- | ------------ |
| Мин. сумма | `YOOKASSA_MIN_AMOUNT_KOPEKS` | 5000 (50 руб) |
| Макс. сумма | `YOOKASSA_MAX_AMOUNT_KOPEKS` | 1000000 (10000 руб) |

## CryptoBot

```env
CRYPTOBOT_TOKEN=your_token
CRYPTOBOT_CURRENCY=RUB
```

Поддерживаемые криптовалюты: USDT, TON, BTC, ETH. Курс конвертации обновляется автоматически через API CryptoBot.

Webhook: `https://hooks.domain.com/cryptobot-webhook`

## Telegram Stars

```env
TELEGRAM_STARS_ENABLED=true
STARS_TO_KOPEKS_RATE=130
```

Курс `130` означает 1 звезда = 1.30 руб. Работает без внешних webhook — обрабатывается через Telegram Bot API.

## CloudPayments

```env
CLOUDPAYMENTS_ENABLED=true
CLOUDPAYMENTS_PUBLIC_ID=pk_xxxxxxxxxxxxx
CLOUDPAYMENTS_API_SECRET=your_api_secret
CLOUDPAYMENTS_CURRENCY=RUB
```

Поддерживает 3D-Secure. Webhook (Pay/Fail уведомления): `https://hooks.domain.com/cloudpayments-webhook`

## Freekassa

```env
FREEKASSA_ENABLED=true
FREEKASSA_SHOP_ID=12345
FREEKASSA_API_KEY=your_api_key
FREEKASSA_SECRET_WORD_1=secret1
FREEKASSA_SECRET_WORD_2=secret2
FREEKASSA_USE_API=true
FREEKASSA_PAYMENT_SYSTEM_ID=44
```

- `FREEKASSA_USE_API=true` — использовать API для создания заказов (рекомендуется для NSPK)
- `FREEKASSA_USE_API=false` — редирект на платежную форму
- `FREEKASSA_PAYMENT_SYSTEM_ID=44` — для NSPK СБП (быстрее и надёжнее)

Webhook: `https://hooks.domain.com/freekassa-webhook`

## Platega

```env
PLATEGA_ENABLED=true
PLATEGA_MERCHANT_ID=your_merchant_id
PLATEGA_SECRET=your_secret_key
PLATEGA_RETURN_URL=https://your-domain.com/payments/success
PLATEGA_FAILED_URL=https://your-domain.com/payments/failed
PLATEGA_ACTIVE_METHODS=2,10,11
PLATEGA_MIN_AMOUNT_KOPEKS=100
PLATEGA_MAX_AMOUNT_KOPEKS=5000000
```

В кабинете Platega укажите webhook: `https://hooks.domain.com/platega-webhook`

## Kassa AI

Отдельная платёжная система (api.fk.life), работает параллельно с Freekassa.

```env
KASSA_AI_ENABLED=true
KASSA_AI_SHOP_ID=12345
KASSA_AI_API_KEY=your_api_key
KASSA_AI_SECRET_WORD_2=secret2
KASSA_AI_PAYMENT_SYSTEM_ID=44
```

- `KASSA_AI_PAYMENT_SYSTEM_ID=44` — СБП QR, `36` — Карты РФ, `43` — SberPay

Webhook: `https://hooks.domain.com/kassa-ai-webhook`
