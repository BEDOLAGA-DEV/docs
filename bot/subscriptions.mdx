## Режимы продаж

| Режим | `SALES_MODE` | Описание |
| ----- | ------------ | -------- |
| Конструктор | `classic` | Пользователь выбирает период, трафик, серверы, устройства |
| Тарифы | `tariffs` | Готовые тарифные планы с фиксированными параметрами |

### Конструктор (по умолчанию)

Пользователь самостоятельно выбирает параметры:
- Период (настраивается через `PERIOD_PRICES`)
- Объём трафика
- Серверы
- Количество устройств

```env
SALES_MODE=classic
PERIOD_PRICES={"30": 200, "90": 500, "180": 900, "365": 1500}
```

### Тарифный режим

Готовые пакеты без сложного выбора. Тарифы настраиваются в админ-панели бота.

```env
SALES_MODE=tariffs
```

Каждый тариф содержит: название, период, трафик, устройства, серверы, цену. Поддерживает скидки промогрупп.

## Режимы продажи трафика

### Выбираемые пакеты (по умолчанию)

Пользователь выбирает из списка пакетов.

```env
TRAFFIC_SELECTION_MODE=selectable
TRAFFIC_PACKAGES_CONFIG="5:2000:false,10:3500:false,50:11000:true,100:15000:true,0:20000:true"
```

Формат: `ГБ:цена_копейки:показать_безлимит`. `0` ГБ означает безлимит.

### Фиксированный лимит

Все подписки с одинаковым объёмом трафика.

```env
TRAFFIC_SELECTION_MODE=fixed
FIXED_TRAFFIC_LIMIT_GB=100
```

`0` означает безлимит.

## Система ценообразования

Цена подписки рассчитывается по формуле:

**Базовая цена + Стоимость трафика + Доп. устройства + Доп. серверы**

Пример расчёта для подписки на 180 дней:

| Компонент | Расчёт | Итого |
| --------- | ------ | ----- |
| Базовый период | — | 400 руб |
| Трафик безлимит | 200 руб/мес x 6 мес | 1200 руб |
| 4 устройства | 50 руб/мес x 6 мес | 300 руб |
| 2 сервера | 100 руб/мес x 6 мес | 1200 руб |
| **Итого** | | **3100 руб** |

## Управление устройствами

```env
TRIAL_DEVICE_LIMIT=1          # Устройства в триал подписке
DEFAULT_DEVICE_LIMIT=3         # Устройства в платной подписке
MAX_DEVICES_LIMIT=15           # Максимум для покупки (0 = без лимита)
```

## Тестовая подписка (триал)

```env
TRIAL_PAYMENT_ENABLED=false    # Требовать оплату за активацию
TRIAL_ACTIVATION_PRICE=0       # Стоимость активации (копейки)
TRIAL_DEVICE_LIMIT=1
```

Триал включает:
- Настраиваемый период и параметры
- Уведомления об истечении
- Автоконверсию в платную подписку
- Контроль обязательной подписки на канал

## Сквозная корзина

Система сохраняет выбранные параметры при недостатке баланса:

1. Пользователь выбирает подписку
2. Баланса не хватает — корзина сохраняется в Redis (TTL 1 час)
3. Пользователь пополняет баланс
4. Кнопка "Вернуться к оформлению" восстанавливает корзину
5. Подписка оформляется автоматически

<Note>
  Для работы корзины требуется Redis: `REDIS_URL=redis://redis:6379/0`
</Note>

## Автопродление

Бот автоматически продлевает подписки при наличии средств на балансе:

- Проверка запускается фоновым сервисом мониторинга
- За N дней до истечения — уведомление пользователю
- При наличии средств — автоматическое продление
- При недостатке — уведомление о необходимости пополнения

## Промо-коды

| Тип | Описание |
| --- | -------- |
| Деньги | Начисляет сумму на баланс |
| Дни подписки | Продлевает подписку на N дней |
| Триал | Выдаёт пробную подписку |
| Промогруппа | Назначает промогруппу со скидками |

## Промогруппы и скидки

Система промогрупп с индивидуальными скидками:

- Скидки на серверы, трафик и устройства
- Скидочные уровни за траты (система лояльности)
- Автоматическое повышение уровня при достижении порога
- Скидки за длительные периоды подписки
