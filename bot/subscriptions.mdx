## Режимы продаж

Бот поддерживает два режима продажи подписок:

### Конструктор (по умолчанию)

Пользователь самостоятельно выбирает параметры подписки:
- Период (настраивается через `PERIOD_PRICES`)
- Объём трафика
- Серверы
- Количество устройств

### Тарифный режим

Предустановленные тарифные планы с фиксированными параметрами. Управляется через админ-панель.

## Конфигурация цен

Цены настраиваются через переменную `PERIOD_PRICES` в формате JSON:

```env
PERIOD_PRICES={"30": 200, "90": 500, "180": 900, "365": 1500}
```

Ключ — количество дней, значение — цена в рублях.

## Сквозная корзина

Система сохраняет выбранные параметры подписки, если у пользователя недостаточно средств:

1. Пользователь выбирает подписку
2. Баланса не хватает — корзина сохраняется в Redis (TTL 1 час)
3. Пользователь пополняет баланс
4. Кнопка "Вернуться к оформлению" восстанавливает корзину
5. Подписка оформляется автоматически

<Info>
  При включённой автопокупке подписка оформляется сразу после пополнения баланса без дополнительных действий.
</Info>

## Автопродление

Бот автоматически продлевает подписки при наличии средств на балансе:

- Проверка запускается фоновым сервисом мониторинга
- За N дней до истечения пользователь получает уведомление
- При наличии средств подписка продлевается автоматически
- При недостатке средств — уведомление о необходимости пополнения

## Промо-коды

Администраторы могут создавать промо-коды через админ-панель:

| Тип | Описание |
| --- | -------- |
| Деньги | Начисляет указанную сумму на баланс |
| Дни подписки | Продлевает активную подписку на N дней |
| Триал | Выдаёт пробную подписку |
