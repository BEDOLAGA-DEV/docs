## Обзор

Админ-панель доступна прямо в Telegram боте для пользователей, чьи Telegram ID указаны в `ADMIN_IDS`.

## Разделы

### Пользователи
- Просмотр списка пользователей
- Поиск по ID, username, имени
- Изменение баланса (начисление/списание)
- Блокировка/разблокировка
- Просмотр подписок и транзакций

### Платежи
- Список всех транзакций
- Фильтрация по провайдеру, статусу, периоду
- Проверка статуса платежа в YooKassa/других провайдерах
- Статистика по провайдерам

### Подписки
- Активные подписки
- Просроченные подписки
- Ручное управление (продление, отключение)

### Рассылки
- Массовые уведомления всем пользователям
- Фильтры: активные подписки, неактивные, все
- Поддержка текста, фото, видео
- Статистика доставки

### Промо-коды
- Генерация промо-кодов
- Типы: деньги, дни подписки, триал
- Лимиты использования
- Статистика активаций

### Бэкапы
- Создание резервной копии БД
- Восстановление из бэкапа
- Автоматические бэкапы по расписанию
- Уведомления в топики о бэкапах

### Мониторинг
- Состояние серверов Remnawave
- Использование памяти и CPU
- Статистика трафика
- Управление режимом тех. работ

### Конкурсы
- Ежедневные игры с призами
- Реферальные конкурсы
- Статистика участников

## Уведомления в топики

Бот отправляет уведомления в Telegram топики (форум):

| Событие | Описание |
| ------- | -------- |
| Активация триала | Новый триал-пользователь |
| Покупка подписки | Новая покупка |
| Конверсия из триала | Триал → платная подписка |
| Продление | Автоматическое продление |
| Пополнение баланса | Новый платёж |
| Тех. работы | Включение/выключение |
| Новая версия | Обнаружена новая версия бота |
