## Что такое Bedolaga Bot

Bedolaga Bot — Telegram-бот для продажи и управления VPN-подписками через панель Remnawave. Включает полный цикл: от регистрации пользователя до аналитики и поддержки.

## Для пользователей

### Покупка и управление подпиской

- **Два режима продаж**: [конструктор и тарифы](/bot/subscriptions) — гибкий выбор или готовые планы
- **Выбор параметров**: период (14-360 дней), трафик (ГБ или безлимит), серверы, устройства
- **Динамический калькулятор** стоимости с учётом скидок
- **Сквозная корзина** — сохранение параметров при недостатке баланса
- **Простая покупка** — упрощённый режим для быстрого оформления
- **Автопродление** — автоматическое списание с баланса
- **Модем-режим** — дополнительные устройства для роутеров

### Платежи

[12 платёжных систем](/bot/payments): Telegram Stars, YooKassa, CryptoBot, Tribute, Heleket, MulenPay, Pal24, Platega, WATA, Freekassa, CloudPayments и ручная оплата. Быстрое пополнение с кнопками сумм.

### Тестовая подписка

- Настраиваемый триал с welcome-цепочкой
- Платный или бесплатный триал
- Автовыдача бонусов за кампании и инвайты
- Контроль подписки на обязательный канал

### Бонусы и скидки

- **[Промокоды](/bot/promo-system)** — 5 типов: деньги, дни, триал, промо-группа, скидка
- **[Промо-группы](/bot/promo-system#промо-группы)** — тарифные уровни с индивидуальными скидками
- **Система лояльности** — автоматическое повышение уровня при достижении порога трат
- **[Персональные предложения](/bot/promo-system#персональные-промо-предложения)** — адресные акции от администратора
- **Тестовый доступ к серверам** — временная выдача через промо-акции

### Реферальная программа

[Полная партнёрская система](/bot/referral-program): персональные ссылки, QR-коды, комиссии с каждого пополнения и покупки, аналитика доходов, вывод средств.

### Конкурсы и игры

[7 типов мини-игр](/bot/contests) с мгновенными призами и реферальные конкурсы с рейтингами.

### Поддержка

[Тикет-система](/bot/support-system) с вложениями, SLA-мониторингом и модераторами. FAQ, правила, оферта.

### Личный кабинет

Полноценный [Cabinet WebApp](/cabinet/overview) внутри Telegram: управление подпиской, платежи, промокоды, устройства, реферальная статистика, FAQ.

## Для администраторов

### Управление пользователями

- Поиск по ID, имени, username, Telegram ID
- Изменение баланса, лимитов устройств, трафика, серверов
- Сброс HWID, перегенерация подписки
- Назначение промо-групп и тарифов
- Покупка подписки пользователю из админки
- Продление/сокращение срока (±365 дней)
- Блокировки с таймером, система ограничений
- Отправка сообщения конкретному пользователю
- Расширенные фильтры: баланс, траты, трафик, активность, кампания
- Чёрный список с управлением

### Аналитика

- Дашборды по пользователям, подпискам и трафику
- Детализация платежей по провайдерам
- Продажи по тарифам, устройствам и странам
- Эффективность кампаний, промокодов и UTM
- Статистика по промо-группам и уровням
- Топ рефереров с аналитикой
- Мониторинг подозрительного трафика

### Продажи и маркетинг

- **[Рекламные кампании](/bot/campaigns)** — deeplink-инвайты с 4 типами бонусов
- **[Промокоды](/bot/promo-system)** — генерация и управление
- **[Промо-группы](/bot/promo-system#промо-группы)** — скидочные уровни
- **[Персональные предложения](/bot/promo-system#персональные-промо-предложения)** — тестовый доступ и скидки
- **[Рассылки](/bot/notifications)** — по сегментам с медиа и кнопками
- **Кастомные кнопки** для рассылок
- **[Конкурсы](/bot/contests)** — ежедневные игры и реферальные конкурсы

### Поддержка и модерация

- **[Тикет-центр](/bot/support-system)** с SLA и приоритетами
- Роли модераторов с ограниченным доступом
- Журнал аудита всех действий
- Блокировки нарушителей с таймером

### Уведомления

- **[Топики для событий](/bot/notifications#топик-уведомления-для-администраторов)** (покупки, триалы, техработы)
- Ежедневные отчёты по расписанию
- Настройка уведомлений и расписаний
- Управление контентом (правила, оферта, FAQ)

### Обслуживание

- **[Бэкапы](/bot/maintenance#бэкапы)** — автоматические и ручные с отправкой в Telegram
- **[Режим техработ](/bot/maintenance#режим-техработ)** — ручной и автоматический
- **[Бан-система](/bot/maintenance#система-бан-интеграции-bedolagaban)** — интеграция с BedolagaBan
- **[Чёрный список](/bot/maintenance#чёрный-список)** — глобальный из GitHub
- **[Ротация логов](/bot/maintenance#ротация-логов)** — автоочистка и архивация
- Мониторинг серверов (XrayChecker)
- Автосинхронизация с Remnawave

### Биллинг и настройки

- Управление ценами без перезапуска
- Управление пакетами трафика
- Тестовые платежи для каждого провайдера
- Режим тарифов и простая покупка
- Управление настройками из бота

### Remnawave

- Синхронизация пользователей (ручная и автоматическая)
- Синхронизация сквадов/серверов
- Управление нодами/сквадами из бота
- Детальная статистика по нодам
- Миграция пользователей между сквадами

### REST API

**FastAPI Web API** с эндпоинтами для управления всеми аспектами бота:
- Управление API-ключами и токенами
- Подписки, пользователи, транзакции
- Промо-система и рассылки
- Контент и настройки

## Автоматизация

| Функция | Описание |
| ------- | -------- |
| Мониторинг Remnawave | Регулярная проверка API, автотехработы |
| Автосинхронизация серверов | Фоновая синхронизация по расписанию |
| Умная корзина | Сохранение параметров в Redis при недостатке баланса |
| Антифрод | Валидация подписки на канал |
| Защита от фишинга | Автоблокировка подозрительных никнеймов |
| Автопроверка транзакций | Проверка статусов "в ожидании" за 24ч |
| Ротация логов | Автоочистка и архивация |
| Волны возврата | Многоступенчатые скидки для истёкших подписок |
