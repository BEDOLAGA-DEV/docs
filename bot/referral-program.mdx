## Обзор

Реферальная программа позволяет пользователям приглашать других и получать бонусы за привлечённых клиентов.

## Включение/отключение

```env
REFERRAL_PROGRAM_ENABLED=true
```

При отключении:
- Кнопки "Партнёрка", "Мои рефералы" скрываются из меню
- Административные функции остаются доступными
- API учитывает настройку

## Что скрывается при отключении

- Кнопка "Партнёрка" в главном меню
- Кнопка "Мои рефералы"
- Другие элементы со словами "partner", "referr", "партнер", "реферал"

## Настройка бонусов

| Переменная | Описание | По умолчанию |
| ---------- | -------- | ------------ |
| `REFERRAL_BONUS_KOPEKS` | Бонус рефереру при регистрации реферала (копейки) | `0` |
| `REFERRAL_BONUS_DAYS` | Бонусные дни подписки рефереру | `0` |

## Механика

1. Пользователь получает персональную реферальную ссылку и QR-код
2. Приглашённый переходит по ссылке и регистрируется
3. Реферер получает бонус (деньги на баланс или дни подписки)
4. Статистика доступна в разделе "Партнёрка"

## Реферальные конкурсы

Администраторы могут создавать конкурсы через API:

```bash
POST /contests/referral
{
  "title": "Рефералы декабрь",
  "contest_type": "referral_paid",
  "start_at": "2025-12-20T10:00:00+03:00",
  "end_at": "2025-12-27T10:00:00+03:00",
  "prize_text": "1 место — 5000 руб, 2 место — 3000 руб"
}
```

Подробнее в [API Reference](/development/api-reference).
