## Что такое Cabinet?

**Bedolaga Cabinet** — это веб-приложение (React + TypeScript), которое предоставляет пользователям интерфейс для управления VPN подписками через браузер.

## Возможности

- Авторизация через Telegram
- Просмотр активных подписок
- Пополнение баланса
- Покупка и продление подписок
- Просмотр истории транзакций
- Мультиязычность (RU/EN)
- Адаптивный дизайн

## Архитектура

```
Браузер  →  Caddy/Nginx  →  /api/*  →  remnawave_bot:8080  (backend API)
                          →  /*      →  /srv/cabinet         (статика frontend)
```

Frontend — статические файлы (HTML, JS, CSS). Обратный прокси:
1. Раздаёт статику frontend
2. Проксирует `/api/*` на backend бота (с удалением префикса `/api`)
